# Todo App

A modern Todo application built using React and Tailwind CSS.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Dark mode toggle
* Form validation
* Responsive design
* Clean and modern user interface

## Technologies Used

* React
* Tailwind CSS
* JavaScript
* Vite

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Run the development server

```bash
npm run dev
```
